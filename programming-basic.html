Python Programming Basics

Introduction

This page covers the basics of Python programming, including variables, data types, control flow, loops, functions, and classes. Each section includes detailed explanations and code examples.

1. Variables and Data Types

Variables are used to store data in a program. Python supports several data types including integers, floats, strings, and booleans.


# Variables and Data Types
integer_var = 10              # Integer
float_var = 20.5              # Float
string_var = "Hello, World!" # String
boolean_var = True            # Boolean

print("Integer:", integer_var)
print("Float:", float_var)
print("String:", string_var)
print("Boolean:", boolean_var)

2. Lists

Lists are used to store multiple items in a single variable. Lists are ordered, mutable, and allow duplicate elements.


# Lists
my_list = [1, 2, 3, 4, 5]    # List of integers
print("\nList:", my_list)

# Adding an item to the list
my_list.append(6)
print("List after appending 6:", my_list)

# Accessing items
print("First item:", my_list[0])

3. Dictionaries

Dictionaries store data in key-value pairs. They are unordered, mutable, and do not allow duplicate keys.


# Dictionaries
my_dict = {
    "name": "Alice",
    "age": 25,
    "city": "New York"
}

print("\nDictionary:", my_dict)

# Accessing items
print("Name:", my_dict["name"])

# Adding a new key-value pair
my_dict["email"] = "alice@example.com"
print("Dictionary after adding email:", my_dict)

4. Control Flow

Control flow statements allow you to control the execution of code based on certain conditions. Python uses if, elif, and else statements for conditional execution.


# Control Flow
x = 15

if x > 10:
    print("\nX is greater than 10")
elif x == 10:
    print("X is equal to 10")
else:
    print("X is less than 10")

5. Loops

Loops are used to execute a block of code repeatedly. Python provides two main types of loops: for loops and while loops.


# Loops
print("\nCounting from 1 to 5:")
for i in range(1, 6):
    print(i)

6. Functions

Functions are blocks of code designed to perform a specific task. Functions help in reusability and organization of code.


# Functions
def greet(name):
    return f"Hello, {name}!"

print("\nFunction output:", greet("Bob"))

7. Classes and Objects

Classes are used to create user-defined data structures. They bundle data and functionality together. Objects are instances of classes.


# Classes and Objects
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        return f"Hi, I'm {self.name} and I'm
         {self.age} years old."

person1 = Person("Charlie", 30)
print("\nClass output:", person1.greet())
